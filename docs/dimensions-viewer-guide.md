## 💡 使用场景

### 场景 1: 浏览所有维度
1. 进入 SCF 维度页面

3. 了解各维度基本信息

1. 点击"核心"查看核心维度
2. 点击"分类"查看分类维度
3. 点击"评估"查看评估维度

## 🔧 API 端点

### 获取维度数据
```
GET /api/knowledge/dimension/{dimensionId}
```

支持的 dimensionId:
- `controls` - 安全控制
- `domains` - 控制域
- `assessment` - 评估目标
- `evidence` - 证据请求
- `sources` - 权威来源
- `privacy` - 隐私原则
- `risk` - 风险目录
- `threat` - 威胁目录
- `lists` - 参考列表

### 响应示例
```json
{
  "dimension": "controls",
  "file": "scf-20254.json",
  "totalRecords": 1451,
  "fields": ["SCF #", "SCF Control", "SCF Domain", ...],
  "records": [...前100条记录...]
}
```
