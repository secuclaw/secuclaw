## 🚀 快速开始

### 第一步：访问功能
```
浏览器访问: http://localhost:5100
导航路径: 知识图谱 → 分类展现
```

### 第二步：选择分类
- **SCF 控制域** - 查看安全域分类
- **MITRE 战术** - 查看攻击战术
- **合规框架** - 查看合规标准
- **优先级** - 查看重要性级别

### 第三步：浏览与搜索
- 使用搜索框快速定位
- 点击卡片查看详情
- 查看统计数据
- 跳转相关内容
