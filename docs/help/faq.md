# Frequently Asked Questions

## General

### What is SecuClaw?

SecuClaw is an AI-driven enterprise security operations platform that combines 20 years of security expertise with cutting-edge AI technology to provide comprehensive security operations capabilities.

### How does SecuClaw differ from OpenClaw?

OpenClaw is a general-purpose AI agent gateway for chat applications. SecuClaw is specifically designed for enterprise security operations, with specialized security agents, threat intelligence integration, compliance management, and SOAR automation.

### What AI models does SecuClaw support?

SecuClaw supports multiple LLM providers:
- Anthropic Claude (recommended)
- OpenAI GPT
- Ollama (local deployment)
- Azure OpenAI
- And more via providers

## Installation

### What are the prerequisites?

- Node.js 22+
- npm, pnpm, or bun
- API key for LLM provider (Anthropic recommended)

### Can I run SecuClaw locally?

Yes! SecuClaw can run locally with:
- Docker (recommended)
- Node.js directly
- Binary installation

## Security

### Is my data secure?

Yes. SecuClaw provides:
- Encryption at rest
- TLS transport
- Secure credential storage
- Session isolation

### Can I deploy on-premises?

Yes! SecuClaw is designed for self-hosted deployment, giving you full control over your security data.

## Features

### What security roles are available?

- SEC (Security Expert)
- SEC+LEG (Privacy Security Officer)
- SEC+IT (Security Architect)
- SEC+BIZ (Business Security Officer)
- CSO (Chief Security Architect)
- SCSO (Supply Chain Security Officer)
- BSO (Business Security Operations)

### What compliance frameworks are supported?

- SCF 2025
- SOC 2
- ISO 27001
- GDPR
- And more

## Pricing

### Is SecuClaw free?

SecuClaw has both open-source and commercial versions:
- **Open Source**: MIT License - core components
- **Enterprise**: Commercial license with support
