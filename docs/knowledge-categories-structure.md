## 🎯 使用场景

### 场景 1: 新手入门
```
1. 访问知识图谱 → 分类展现
2. 选择"优先级"维度
3. 查看"高优先级"分类
4. 了解 287 个关键控制
```

### 场景 2: 合规审计
```
1. 选择"合规框架"维度
2. 选择目标框架（例如: GDPR）
3. 查看 145 个相关控制
4. 确认合规覆盖情况
```

### 场景 3: 威胁分析
```
1. 选择"MITRE 战术"维度
2. 浏览各攻击阶段
3. 选择高发战术（例如: TA0005）
4. 了解 98 个防御规避技术
```

### 场景 4: 安全规划
```
1. 选择"SCF 控制域"维度
2. 识别关键域（例如: NET 网络安全）
3. 查看该域的 98 个控制
4. 制定实施计划
```
