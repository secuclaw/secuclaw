## 输出要求

### 必须产出的文件

1. **trait.ts** - Tool Trait定义
   ```typescript
   // 工具能力接口
   export interface IToolCapabilities {
     readonly destructive: boolean;      // 是否有破坏性
     readonly networkAccess: boolean;    // 是否需要网络
     readonly filesystemAccess: boolean; // 是否访问文件系统
     readonly requiresApproval: boolean; // 是否需要批准
     readonly estimatedTime: number;     // 预估执行时间(ms)
   }
   
   // 工具主接口
   export interface ITool<TParams = unknown, TResult = unknown> {
     readonly id: string;
     readonly name: string;
     readonly description: string;
     readonly category: ToolCategory;
     readonly capabilities: IToolCapabilities;
     
     // Schema
     getSchema(): ToolSchema;
     validateParams(params: unknown): params is TParams;
     
     // 执行
     execute(params: TParams, context: ToolContext): Promise<ToolResult<TResult>>;
     
     // 生命周期
     initialize?(): Promise<void>;
     dispose?(): Promise<void>;
     
     // 帮助
     getHelp(): string;
     getExamples(): ToolExample[];
   }
   
   // 工具上下文
   export interface IToolContext {
     sessionId: string;
     agentId: string;
     userId: string;
     workspaceDir?: string;
     sandbox?: ISandbox;
     permissions: string[];
     metadata?: Record<string, unknown>;
   }
   
   // 工具结果
   export interface IToolResult<T = unknown> {
     success: boolean;
     data?: T;
     error?: ToolError;
     metadata?: Record<string, unknown>;
     duration?: number;
   }
   ```

2. **base.ts** - 基类实现（重构）
   - `BaseTool<TParams, TResult>` 抽象类
   - 通用参数验证
   - 通用错误处理
   - 通用超时处理
   - 执行日志

3. **schema.ts** - Schema定义（重构）
   - `ToolSchemaBuilder` 类
   - `validateSchema()` - Schema验证
   - `inferSchema()` - 从类型推断Schema
   - 支持JSON Schema格式

4. **executor.ts** - 执行器（重构）
   - `ToolExecutor` 类
   - `execute()` - 执行工具
   - `executeWithTimeout()` - 带超时执行
   - `executeInSandbox()` - 沙箱执行
   - 批量执行支持

5. **registry.ts** - 注册表（重构）
   - `ToolRegistry` 类
   - `register()` - 注册工具
   - `get()` - 获取工具
   - `list()` - 列出工具
   - `getByCategory()` - 按类别获取
   - `search()` - 搜索工具

6. **测试文件**
   - trait.test.ts
   - base.test.ts
   - schema.test.ts
   - executor.test.ts
   - registry.test.ts
