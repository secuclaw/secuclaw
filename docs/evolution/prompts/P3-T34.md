## 示例代码片段

### index.ts 骨架

```typescript
import { CommandExecutor } from './executor';
import { Sandbox } from './sandbox';
import { WhitelistManager } from './whitelist';

export class EdgeShellTool {
  private executor: CommandExecutor;
  private sandbox: Sandbox;
  private whitelist: WhitelistManager;

  constructor() {
    this.executor = new CommandExecutor();
    this.sandbox = new Sandbox();
    this.whitelist = new WhitelistManager();
  }

  /**
   * 执行命令
   */
  async execute(command: string, args: string[] = []): Promise<ShellResult> {
    // 检查白名单
    if (!this.isAllowed(command)) {
      throw new Error(`Command not allowed: ${command}`);
    }

    // 执行命令
    return await this.executor.execute(command, args);
  }

  /**
   * 检查白名单
   */
  isAllowed(command: string): boolean {
    return this.whitelist.check(command);
  }

  /**
   * 沙箱执行
   */
  async executeSandboxed(command: string): Promise<ShellResult> {
    return await this.sandbox.execute(command);
  }
}
```

## 注意事项

1. **白名单**: 只允许安全的命令
2. **超时控制**: 防止命令无限执行
3. **资源清理**: 执行后清理临时文件
4. **日志记录**: 记录所有命令执行
