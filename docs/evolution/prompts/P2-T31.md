## 任务描述

实现Merkle树审计追踪系统，为每个操作生成密码学链接的审计记录。该系统确保审计日志的不可篡改性，支持取证回溯和完整性验证。

## 输出要求

### 必须产出的文件

1. **tree.ts** - Merkle树实现
   ```typescript
   export class MerkleTree {
     // 构建树
     build(leaves: Buffer[]): MerkleNode;
     
     // 获取根哈希
     getRoot(): Buffer;
     
     // 生成证明
     getProof(leaf: Buffer): MerkleProof;
     
     // 验证证明
     verifyProof(proof: MerkleProof, root: Buffer): boolean;
     
     // 添加叶子
     addLeaf(data: Buffer): number;
     
     // 批量添加
     addLeaves(data: Buffer[]): number[];
   }
   ```

2. **node.ts** - 节点定义
   - `MerkleNode` 类
   - `createNode()` - 创建节点
   - `hashNode()` - 计算节点哈希
   - `isLeaf()` - 判断叶子节点
   - `getChildren()` - 获取子节点

3. **proof.ts** - 证明验证
   - `MerkleProof` 类
   - `generateProof()` - 生成证明
   - `verifyProof()` - 验证证明
   - `serializeProof()` - 序列化证明
   - `deserializeProof()` - 反序列化证明

4. **hash.ts** - 哈希函数
   - `sha256()` - SHA-256哈希
   - `hashPair()` - 哈希节点对
   - `hashLeaf()` - 哈希叶子
   - `hashData()` - 哈希数据

5. **chain.ts** - 审计链
   - `AuditChain` 类
   - `append()` - 追加条目
   - `getEntry()` - 获取条目
   - `verifyChain()` - 验证链完整性
   - `getProof()` - 获取条目证明
   - `exportChain()` - 导出链
   - `importChain()` - 导入链

6. **entry.ts** - 审计条目
   - `AuditEntry` 类
   - `createEntry()` - 创建条目
   - `signEntry()` - 签名条目
   - `verifyEntry()` - 验证条目
   - `serializeEntry()` - 序列化条目

7. **verifier.ts** - 验证器
   - `AuditVerifier` 类
   - `verifyEntry()` - 验证单个条目
   - `verifyRange()` - 验证范围
   - `verifyChain()` - 验证整链
   - `detectTampering()` - 检测篡改

8. **storage.ts** - 持久化
   - `MerkleStorage` 类
   - `save()` - 保存树
   - `load()` - 加载树
   - `append()` - 追加叶子
   - `getProof()` - 获取证明

9. **types.ts** - 类型定义
   - `MerkleNode` - 节点类型
   - `MerkleProof` - 证明类型
   - `AuditEntry` - 审计条目类型
   - `AuditChain` - 审计链类型
   - `HashFunction` - 哈希函数类型

10. **测试文件**
    - tree.test.ts
    - proof.test.ts
    - chain.test.ts
    - verifier.test.ts
    - storage.test.ts
