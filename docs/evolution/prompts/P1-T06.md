## 输出要求

### 必须产出的文件

1. **manager.ts** - 渠道管理器
   ```typescript
   export class ChannelManager {
     register(channel: BaseChannel): void;
     unregister(channelId: string): void;
     get(channelId: string): BaseChannel | undefined;
     getAll(): BaseChannel[];
     getByType(type: string): BaseChannel[];
     getStatus(channelId: string): ChannelStatus;
     getMetrics(channelId: string): ChannelMetrics;
     broadcast(message: UnifiedMessage): Promise<void>;
     healthCheckAll(): Promise<Map<string, HealthResult>>;
   }
   ```

2. **registry.ts** - 渠道注册表
   ```typescript
   export class ChannelRegistry {
     registerFactory(type: string, factory: ChannelFactory): void;
     unregisterFactory(type: string): void;
     create(type: string, config: ChannelConfig): BaseChannel;
     getSupportedTypes(): string[];
     getFactory(type: string): ChannelFactory | undefined;
   }
   ```

3. **base.ts** - 渠道基类增强
   ```typescript
   export abstract class BaseChannel {
     abstract connect(): Promise<void>;
     abstract disconnect(): Promise<void>;
     abstract sendMessage(msg: UnifiedMessage): Promise<string>;
     abstract onMessage(handler: MessageHandler): void;
     
     // 新增方法
     getStatus(): ChannelStatus;
     getMetrics(): ChannelMetrics;
     healthCheck(): Promise<HealthResult>;
     startTyping?(chatId: string): Promise<void>;
     stopTyping?(chatId: string): Promise<void>;
   }
   ```

4. **status-monitor.ts** - 状态监控
   - `StatusMonitor` 类
   - `startMonitoring()` - 开始监控
   - `stopMonitoring()` - 停止监控
   - `onStatusChange()` - 状态变更回调

5. **types.ts** - 统一类型
   - `ChannelStatus` - 状态枚举
   - `ChannelMetrics` - 指标类型
   - `UnifiedMessage` - 统一消息格式
   - `HealthResult` - 健康检查结果

6. **manager.test.ts** - 单元测试

## 设计要点

### 渠道生命周期

```
┌─────────┐     ┌─────────┐     ┌─────────┐     ┌─────────────┐
│Created  │ ──► │Connected│ ──► │  Ready  │ ──► │Disconnected │
└─────────┘     └─────────┘     └─────────┘     └─────────────┘
     │               │               │               │
     └───────────────┴───────────────┴───────────────┘
                        Error → Reconnect
```

### 统一消息格式

```typescript
interface UnifiedMessage {
  id: string;
  channelId: string;
  channelType: string;
  from: User;
  to: User | Group;
  content: string | RichContent;
  attachments: Attachment[];
  timestamp: Date;
  metadata: Record<string, any>;
}
```

### 状态监控

```typescript
interface ChannelMetrics {
  messagesReceived: number;
  messagesSent: number;
  errors: number;
  lastActivity?: Date;
  averageLatency: number;
}
```
