## 输出要求

### 必须产出的文件

1. **trait.ts** - Channel Trait定义
   ```typescript
   // Channel能力接口
   export interface IChannelCapabilities {
     readonly text: boolean;
     readonly markdown: boolean;
     readonly attachments: boolean;
     readonly buttons: boolean;
     readonly carousel: boolean;
     readonly typing: boolean;
     readonly reply: boolean;
     readonly edit: boolean;
     readonly delete: boolean;
     readonly maxAttachmentSize: number;
     readonly supportedFormats: string[];
   }
   
   // Channel主接口
   export interface IMessageChannel {
     readonly id: string;
     readonly name: string;
     readonly capabilities: IChannelCapabilities;
     
     // 连接管理
     connect(): Promise<void>;
     disconnect(): Promise<void>;
     isConnected(): boolean;
     
     // 消息发送
     send(message: OutgoingMessage): Promise<MessageResult>;
     sendTyping(chatId: string): Promise<void>;
     
     // 消息接收
     onMessage(handler: MessageHandler): void;
     onError(handler: ErrorHandler): void;
     
     // 消息操作
     edit(chatId: string, messageId: string, text: string): Promise<void>;
     delete(chatId: string, messageId: string): Promise<void>;
     
     // 聊天管理
     getChatInfo(chatId: string): Promise<ChatInfo>;
     getMemberInfo(chatId: string, userId: string): Promise<MemberInfo>;
   }
   
   // 消息处理器
   export type MessageHandler = (message: IncomingMessage) => Promise<void>;
   export type ErrorHandler = (error: ChannelError) => void;
   ```

2. **base.ts** - 基类实现
   - `BaseChannel` 抽象类
   - 通用连接管理
   - 通用错误处理
   - 通用重试逻辑
   - 事件发射器

3. **message.ts** - 消息类型
   - `IncomingMessage` 类型
   - `OutgoingMessage` 类型
   - `MessageResult` 类型
   - `MessageBuilder` 类

4. **attachment.ts** - 附件类型
   - `Attachment` 类型
   - `AttachmentType` 枚举
   - `AttachmentBuilder` 类
   - 支持图片、视频、音频、文件

5. **registry.ts** - 通道注册表
   - `ChannelRegistry` 类
   - `register()` - 注册通道
   - `get()` - 获取通道
   - `list()` - 列出所有通道
   - `getByCapability()` - 按能力获取

6. **factory.ts** - 工厂函数
   - `createChannel()` - 创建通道
   - `createChannelFromConfig()` - 从配置创建

7. **测试文件**
   - trait.test.ts
   - base.test.ts
   - message.test.ts
   - registry.test.ts
