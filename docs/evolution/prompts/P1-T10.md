## 示例代码片段

### base.ts 骨架

```typescript
export interface STTOptions {
  language?: string;
  model?: string;
  punctuation?: boolean;
  profanityFilter?: boolean;
}

export interface TranscriptionResult {
  text: string;
  confidence: number;
  language: string;
  duration: number;
  words?: WordTiming[];
}

export interface TranscriptionChunk {
  text: string;
  isFinal: boolean;
  confidence: number;
}

export abstract class STTProvider {
  abstract getName(): string;
  abstract transcribe(audio: Buffer, options?: STTOptions): Promise<TranscriptionResult>;
  abstract transcribeStream(audio: AsyncIterable<Buffer>, options?: STTOptions): AsyncIterable<TranscriptionChunk>;
  abstract getSupportedLanguages(): string[];
}
```

| Azure STT | 94%+ | 低 | 中 | 企业级 |

## 注意事项

1. 不同提供商支持的音频格式不同
2. 需要处理音频采样率转换
3. 流式识别需要WebSocket支持
4. 本地Whisper需要额外安装whisper.cpp
