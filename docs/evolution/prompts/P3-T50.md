## 任务描述

实现6位一次性配对码机制，用于首次设备连接认证。配对成功后生成设备证书，支持零信任访问控制。

## 输出要求

### 必须产出的文件

1. **code.ts** - 配对码生成
   ```typescript
   export class PairingCode {
     // 生成配对码
     static generate(): string;  // 6位数字
     
     // 验证配对码
     static validate(code: string): boolean;
     
     // 哈希配对码
     static hash(code: string): string;
   }
   
   export class PairingCodeManager {
     // 创建配对会话
     createSession(): PairingSession;
     
     // 验证配对码
     verifyCode(sessionId: string, code: string): boolean;
     
     // 检查过期
     isExpired(sessionId: string): boolean;
     
     // 清理过期会话
     cleanupExpired(): void;
   }
   ```

2. **session.ts** - 配对会话
   - `PairingSession` 类
   - `create()` - 创建会话
   - `expire()` - 过期会话
   - `complete()` - 完成配对
   - 会话超时：5分钟

3. **device.ts** - 设备证书
   - `DeviceCertificate` 类
   - `generate()` - 生成证书
   - `sign()` - 签名证书
   - `verify()` - 验证证书
   - `revoke()` - 撤销证书

4. **manager.ts** - 配对管理器
   - `PairingManager` 类
   - `initiatePairing()` - 发起配对
   - `completePairing()` - 完成配对
   - `listDevices()` - 列出设备
   - `revokeDevice()` - 撤销设备
   - `isPaired()` - 检查配对状态

5. **types.ts** - 类型定义
   - `PairingSession` - 配对会话类型
   - `DeviceCertificate` - 设备证书类型
   - `PairingStatus` - 配对状态枚举
   - `DeviceInfo` - 设备信息

6. **测试文件**
   - code.test.ts
   - session.test.ts
   - device.test.ts
   - manager.test.ts
