## 输出要求

### 必须产出的文件

1. **trait.ts** - Memory Trait定义
   ```typescript
   // 记忆能力接口
   export interface IMemoryCapabilities {
     readonly vectorSearch: boolean;      // 向量搜索
     readonly keywordSearch: boolean;     // 关键词搜索
     readonly hybridSearch: boolean;      // 混合搜索
     readonly persistence: boolean;       // 持久化
     readonly embeddings: boolean;        // 嵌入支持
     readonly compaction: boolean;        // 压缩支持
     readonly maxEntries: number;         // 最大条目数
     readonly maxStorageMb: number;       // 最大存储(MB)
   }
   
   // 记忆主接口
   export interface IMemoryStore {
     readonly id: string;
     readonly name: string;
     readonly capabilities: IMemoryCapabilities;
     
     // 存储
     store(entry: MemoryEntry): Promise<string>;
     storeBatch(entries: MemoryEntry[]): Promise<string[]>;
     
     // 检索
     retrieve(id: string): Promise<MemoryEntry | null>;
     
     // 搜索
     search(query: MemoryQuery): Promise<MemorySearchResult>;
     vectorSearch(embedding: number[], options?: SearchOptions): Promise<MemorySearchResult>;
     keywordSearch(query: string, options?: SearchOptions): Promise<MemorySearchResult>;
     hybridSearch(query: string, embedding?: number[], options?: SearchOptions): Promise<MemorySearchResult>;
     
     // 更新
     update(id: string, entry: Partial<MemoryEntry>): Promise<void>;
     
     // 删除
     delete(id: string): Promise<void>;
     deleteBySession(sessionId: string): Promise<void>;
     
     // 管理
     compact(): Promise<CompactionResult>;
     stats(): Promise<MemoryStats>;
     clear(): Promise<void>;
     
     // 生命周期
     initialize(): Promise<void>;
     dispose(): Promise<void>;
   }
   
   // 嵌入接口
   export interface IEmbeddingProvider {
     embed(text: string): Promise<number[]>;
     embedBatch(texts: string[]): Promise<number[][]>;
   }
   ```

2. **base.ts** - 基类实现（重构）
   - `BaseMemoryStore` 抽象类
   - 通用搜索逻辑
   - 通用压缩逻辑
   - 通用错误处理
   - 缓存层

3. **config.ts** - 配置类型
   - `MemoryConfig` 类型
   - `MemoryConfigBuilder` 类
   - `validateConfig()` - 配置验证
   - 支持多后端配置

4. **registry.ts** - 注册表
   - `MemoryRegistry` 类
   - `register()` - 注册存储
   - `get()` - 获取存储
   - `list()` - 列出存储
   - `getByCapability()` - 按能力获取

5. **factory.ts** - 工厂函数
   - `createMemoryStore()` - 创建存储
   - `createMemoryStoreFromConfig()` - 从配置创建
   - 自动检测最佳后端

6. **测试文件**
   - trait.test.ts
   - base.test.ts
   - registry.test.ts
   - factory.test.ts
