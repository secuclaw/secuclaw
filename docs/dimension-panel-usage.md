# SCF 维度面板 - 快速使用指南

## 访问方式

### 步骤 1: 打开知识库
访问 **http://localhost:5101** (或 http://localhost:5100)

点击左侧导航栏的 **"知识库" 🧠**

### 步骤 2: 打开 SCF 维度面板
在知识库页面右上角，点击 **"SCF维度"** 按钮

右侧将滑入 SCF 数据维度面板（450px 宽）

### 步骤 3: 隐藏维度面板
再次点击 **"隐藏维度"** 按钮，或点击面板右上角的 **X** 按钮

## 功能特性

### 右侧面板内容
- 📊 **9 个数据维度** - 网格/列表视图
- 🔍 **搜索和过滤** - 按类型过滤
- 📋 **维度详情** - 点击卡片查看详情
- 📈 **数据统计** - 记录数、文件大小

### 主内容区域
所有知识图谱功能保持不变：
- 知识图谱
- 攻击链分析
- 控制覆盖
- SCF-MITRE
- 攻击路径
- 深度防御
- 分类展现

## 工作流程示例

### 场景 1: 查看维度数据
1. 进入知识库页面
2. 点击"SCF维度"按钮
3. 查看所有 9 个维度
4. 点击"安全控制"卡片
5. 查看控制详情和数据

### 场景 2: 图谱与维度对比
1. 打开知识图谱视图
2. 点击"SCF维度"打开右侧面板
3. 左侧查看知识图谱
4. 右侧浏览维度数据
5. 对比分析关联关系

### 场景 3: 分类与维度联动
1. 切换到"分类展现"视图
2. 点击"SCF维度"打开面板
3. 左侧选择控制域分类
4. 右侧查看对应的维度数据
5. 双维度交叉分析

## 快捷操作

### 按钮说明
| 按钮 | 功能 |
|------|------|
| **SCF维度** | 打开右侧维度面板 |
| **隐藏维度** | 关闭维度面板 |
| **刷新数据** | 刷新知识图谱数据 |
| **X** | 关闭维度面板 |

### 视图切换
- 知识图谱、攻击链等视图与维度面板独立
- 切换视图不会关闭维度面板
- 可以在任何视图下使用维度面板

## 提示

### 💡 使用技巧
1. **双屏工作** - 左侧看图谱，右侧看维度
2. **按需显示** - 不用时可隐藏面板
3. **独立滚动** - 两侧内容独立浏览
4. **快速切换** - 任何视图都可打开面板

### ⚠️ 注意事项
- 面板宽度固定为 450px
- 面板不支持拖拽调整
- 关闭后状态不保存（刷新页面会重置）
- 移动端可能需要适配

## 常见问题

**Q: 面板在哪里打开？**
A: 进入"知识库"页面后，点击右上角的"SCF维度"按钮

**Q: 如何关闭面板？**
A: 点击"隐藏维度"按钮或面板右上角的 X

**Q: 面板会影响左侧内容吗？**
A: 不会，主内容区域完全独立可用

**Q: 可以调整面板宽度吗？**
A: 当前不支持，固定 450px 宽

**Q: 切换视图后面板会关闭吗？**
A: 不会，面板状态保持
