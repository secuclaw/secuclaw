# 安全控制台

SecuClaw安全控制台是一个基于Web的仪表板，用于监控安全运营、管理事件和配置系统。

## 访问

- **本地**: [http://127.0.0.1:21000/](http://127.0.0.1:21000/)
- **远程**: 配置[远程访问](/zh-CN/gateway/remote)以进行外部访问

## 功能

### 仪表板

主仪表板提供安全态势的实时概览。

**组件：**
- 安全评分摘要
- 活动警报数量
- 最近事件
- 威胁情报摘要
- 合规状态

### 威胁作战室（War Room）

War Room是安全分析师调查和响应威胁的工作区。

**功能：**
- 实时警报流
- 事件调查工作区
- 威胁时间线可视化
- IOC（妥协指标）查询
- MITRE ATT&CK映射

### 合规审计官

合规审计官跟踪组织的合规态势。

**功能：**
- 框架合规状态（SCF 2025、SOC 2、ISO 27001）
- 控制差距分析
- 审计跟踪查看器
- 合规报告
- 修复跟踪

### 风险仪表板

风险仪表板提供企业风险可见性。

**功能：**
- 风险评分趋势
- 资产漏洞矩阵
- 业务影响分析
- 风险登记册
- 处理计划

## 配置

通过Web界面管理您的SecuClaw部署。

**选项卡：**
- **常规**: 网关设置、端口、认证
- **代理**: 模型配置、代理角色
- **安全**: 数据源、威胁情报
- **合规**: 框架设置、报告
- **自动化**: SOAR剧本、触发器
