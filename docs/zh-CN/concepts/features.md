## 完整功能列表

### 安全代理
- **SEC（安全专家）**：一般安全分析和威胁评估
- **SEC+LEG（隐私安全官）**：隐私合规和数据保护
- **SEC+IT（安全架构师）**：基础设施安全架构
- **SEC+BIZ（业务安全官）**：业务风险和安全协调
- **CSO（首席安全架构官）**：企业级安全战略
- **SCSO（供应链安全官）**：供应链安全管理
- **BSO（业务安全运营）**：运营安全管理

### 威胁检测
- 实时安全事件分析
- 使用ML模型进行异常检测
- 行为分析
- IOC（妥协指标）检测
- MITRE ATT&CK框架集成

### 合规管理
- SCF 2025框架支持
- 自动化合规映射
- 控制差距分析
- 审计跟踪生成
- 监管报告（SOC 2、ISO 27001、GDPR等）

### 安全自动化（SOAR）
- 自动化事件响应
- 剧本执行
- 工单创建和管理
- 警报分类自动化
- 取证数据收集

### 威胁情报
- MITRE ATT&CK集成
- STIX 2.1威胁源
- 自定义IOC管理
- 威胁狩猎自动化
- 风险评分

### 风险管理
- 资产漏洞评估
- 风险评分和优先级排序
- 合规风险分析
- 业务影响评估
- 攻击面监控

### 数据源
- SIEM集成
- 防火墙日志
- EDR/XDR集成
- 云安全日志
- 网络流量分析

### 集成能力
- 外部集成REST API
- Webhook支持自动化
- SIEM连接器
- 工单系统集成
- SOAR平台集成
