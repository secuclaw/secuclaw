# 安全角色

SecuClaw提供专业AI安全代理，每个代理都针对特定的安全运营需求设计。

## 角色概览

| 角色 | 代码 | 能力 | 用例 |
|------|------|------|------|
| 安全专家 | SEC | 威胁分析、漏洞评估 | 一般安全运营 |
| 隐私安全官 | SEC+LEG | 隐私合规、数据保护 | GDPR、CCPA合规 |
| 安全架构师 | SEC+IT | 基础设施安全、架构审查 | 云安全、网络安全 |
| 业务安全官 | SEC+BIZ | 风险协调、业务连续性 | 企业风险管理 |
| 首席安全架构官 | CSO | 企业战略、治理 | C级安全领导 |
| 供应链安全官 | SCSO | 供应商风险、第三方安全 | 供应链风险管理 |
| 业务安全运营 | BSO | SOC运营、事件响应 | 日常安全运营 |

## 详细角色能力

### SEC - 安全专家

基础安全代理角色，用于一般安全运营。

**能力：**
- 威胁分析与评估
- 漏洞识别
- 安全事件分析
- 事件调查指导
- 安全最佳实践建议

**适用于：**
- 一般安全咨询
- 漏洞评估
- 安全事件分类
- 风险识别

### SEC+LEG - 隐私安全官

结合安全专业知识与法律合规知识。

**能力：**
- 隐私法规合规（GDPR、CCPA、LGPD）
- 数据保护影响评估
- 隐私政策审查
- 数据泄露响应
- 同意管理

**适用于：**
- 隐私合规问题
- 数据保护评估
- 监管合规
- 隐私政策审查

### SEC+IT - 安全架构师

结合安全专业知识与IT基础设施知识。

**能力：**
- 架构安全审查
- 云安全评估
- 网络安全设计
- 零信任实施
- DevSecOps集成

**适用于：**
- 基础设施安全
- 架构审查
- 云安全
- 网络分段

### SEC+BIZ - 业务安全官

结合安全专业知识与业务协调。

**能力：**
- 业务风险评估
- 安全战略协调
- 安全投资回报分析
- 业务连续性规划
- 第三方风险

**适用于：**
- 高管安全咨询
- 风险优先级排序
- 预算规划
- 业务影响分析

### CSO - 首席安全架构官

企业级安全领导角色。

**能力：**
- 企业安全战略
- 治理框架设计
- 安全政策制定
- 董事会级报告
- 并购安全尽职调查

**适用于：**
- 战略安全规划
- 治理开发
- 高管简报
- 安全转型

### SCSO - 供应链安全官

专注于第三方和供应链安全。

**能力：**
- 供应商风险评估
- 第三方安全审查
- 供应链威胁分析
- 合同安全条款
- 第四方监控

**适用于：**
- 供应商评估
- 供应链安全
- 第三方风险
- 合同审查

### BSO - 业务安全运营

日常安全运营管理。

**能力：**
- SOC告警分类
- 事件响应协调
- 安全监控
- 合规报告
- 团队协调

**适用于：**
- SOC运营
- 事件管理
- 日常安全任务
- 团队领导

## 多角色协作

对于复杂的安全挑战，多个角色可以协作：

```json5
{
  agents: {
    list: [
      { id: "sec", role: "SEC" },
      { id: "legal", role: "SEC+LEG" },
      { id: "arch", role: "SEC+IT" },
    ],
  },
  bindings: [
    { agentId: "sec", match: { channel: "console" } },
    { agentId: "legal", match: { topic: "compliance" } },
    { agentId: "arch", match: { topic: "architecture" } },
  ],
}
```
