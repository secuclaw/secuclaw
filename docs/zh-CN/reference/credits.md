# 关于 SecuClaw

## 项目概览

**SecuClaw**（安爪安全）是一款AI驱动的企业级安全运营平台，融合20年安全行业经验与前沿AI技术。

**使命**：通过智能安全运营赋能企业，实现主动威胁检测、自动化响应和持续合规。

## 基于 OpenClaw

SecuClaw构建于 [OpenClaw](https://github.com/openclaw/openclaw) 开源项目之上，该项目提供了AI代理的核心网关架构。

## 产品线

| 产品 | 描述 |
|---------|-------------|
| SecuClaw Platform | 核心安全运营平台 |
| SecuHub | 安全能力技能市场 |
| SecuClaw Intelligence | 威胁情报服务 |
| SecuClaw Command | 安全运营中心 |
| SecuClaw Compliance | 合规管理 |

## 技术栈

- **运行时**: Node.js 22+
- **语言**: TypeScript
- **AI**: Anthropic Claude, OpenAI GPT, Ollama
- **数据库**: SQLite (JSONL), ChromaDB (向量)
- **容器**: Docker

## 许可证

SecuClaw提供多种许可选项：
