# 常见问题

## 一般问题

### 什么是SecuClaw？

SecuClaw是一款AI驱动的企业级安全运营平台，融合20年安全专业知识和前沿AI技术，提供全面的安全运营能力。

### SecuClaw与OpenClaw有什么区别？

OpenClaw是通用的AI代理网关，用于聊天应用。SecuClaw专门为企业安全运营设计，具有专业安全代理、威胁情报集成、合规管理和SOAR自动化功能。

### SecuClaw支持哪些AI模型？

- Anthropic Claude（推荐）
- OpenAI GPT
- Ollama（本地部署）
- Azure OpenAI

## 安装

### 前置要求是什么？

- Node.js 22+
- npm、pnpm或bun
- LLM提供商的API密钥（推荐Anthropic）

### 我可以在本地运行SecuClaw吗？

可以！SecuClaw可以通过Docker、Node.js直接运行或二进制安装。

## 安全

### 我的数据安全吗？

是的。SecuClaw提供静态加密、TLS传输、安全凭证存储和会话隔离。

### 我可以本地部署吗？

可以！SecuClaw专为自托管部署设计，让您完全掌控安全数据。

## 功能

### 有哪些安全角色可用？

- SEC（安全专家）
- SEC+LEG（隐私安全官）
- SEC+IT（安全架构师）
- SEC+BIZ（业务安全官）
- CSO（首席安全架构官）
- SCSO（供应链安全官）
- BSO（业务安全运营）

### 支持哪些合规框架？

SCF 2025、SOC 2、ISO 27001、GDPR等。

## 定价

### SecuClaw免费吗？

SecuClaw有开源和商业两个版本：
- **开源**: MIT许可证 - 核心组件
- **企业**: 商业许可证，含支持服务
