# 知识图谱 - 分类展现功能

## 📋 功能概述

知识图谱分类展现功能提供了多维度浏览知识库的能力，让用户可以按照不同的分类维度查看 SCF 安全控制和 MITRE ATT&CK 威胁技术。

## 🔍 功能特性

### 搜索功能
- 支持按分类名称搜索
- 支持按描述内容搜索
- 支持按 ID 代码搜索
- 实时显示搜索结果数量

### 分类卡片
- 每个分类以卡片形式展示
- 显示分类名称、ID、描述
- 显示包含的条目数量
- 可视化进度条显示相对大小
- 颜色编码便于识别

### 交互功能
- 点击分类卡片查看详情
- 分类状态高亮显示
- 支持筛选和过滤
- 统计信息实时更新

## 💻 使用方式

### 访问路径
1. 打开 Web 前端: http://localhost:5100
2. 进入"知识图谱"页面
3. 点击"分类展现"选项卡

### 操作步骤

#### 按控制域浏览
1. 选择"SCF 控制域"选项卡
2. 浏览 34 个控制域卡片
3. 点击感兴趣的域查看详情
4. 使用搜索框快速定位特定域

#### 按战术浏览
1. 选择"MITRE 战术"选项卡
2. 查看 10+ 个战术阶段
3. 了解每个战术包含的技术数量
4. 选择战术查看相关技术

#### 按合规框架浏览
1. 选择"合规框架"选项卡
2. 浏览不同合规框架
3. 查看框架映射的控制数量
4. 选择框架查看合规要求

#### 按优先级浏览
1. 选择"优先级"选项卡
2. 查看 4 个优先级级别
3. 了解各级别的控制数量
4. 选择优先级查看相关控制

### 搜索操作
1. 在搜索框输入关键词
2. 系统自动过滤分类
3. 显示匹配结果数量
4. 支持名称、描述、ID 搜索
