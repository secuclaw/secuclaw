# 知识图谱分类展现 - 快速使用指南

## 🚀 快速开始

### 1. 访问分类展现
```
浏览器访问: http://localhost:5100
→ 导航到: 知识图谱
→ 点击: "分类展现" 选项卡
```

### 2. 选择分类维度

在页面顶部选择要查看的分类类型：

- **SCF 控制域** 🔵 - 按安全域查看控制
- **MITRE 战术** 🎯 - 按攻击战术阶段查看
- **合规框架** 📋 - 按合规标准查看
- **优先级** ⚡ - 按控制重要性查看

### 3. 浏览分类

- 浏览卡片式分类展示
- 每个卡片显示：
  - 分类名称和 ID
  - 简短描述
  - 包含的条目数量
  - 颜色编码

### 4. 搜索分类

在搜索框输入关键词：
- 支持分类名称搜索
- 支持描述内容搜索
- 支持 ID 代码搜索
- 实时显示结果数量

### 5. 查看详情

点击任意分类卡片：
- 高亮显示选中的分类
- 显示详细信息
- 可跳转到相关内容

## 💡 使用场景

### 场景 1: 了解特定控制域
```
1. 选择"SCF 控制域"
2. 搜索或找到目标域（例如: GOV - 治理）
3. 点击卡片查看详情
4. 查看该域的 38 个控制
```

### 场景 2: 分析攻击技术
```
1. 选择"MITRE 战术"
2. 选择感兴趣的战术（例如: TA0001 - 初始访问）
3. 查看该战术的 52 个技术
4. 了解相关防御措施
```

### 场景 3: 合规检查
```
1. 选择"合规框架"
2. 选择要检查的框架（例如: GDPR）
3. 查看 145 个相关控制
4. 确保合规覆盖
```

### 场景 4: 优先实施
```
1. 选择"优先级"
2. 重点关注"高优先级"
3. 查看 287 个关键控制
4. 优先实施这些控制
```
