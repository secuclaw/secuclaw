# About SecuClaw

## Project Overview

**SecuClaw** (安爪安全) is an AI-driven enterprise security operations platform that combines 20 years of security industry expertise with cutting-edge AI technology.

**Mission**: Empower enterprises with intelligent security operations, enabling proactive threat detection, automated response, and continuous compliance.

## Based on OpenClaw

SecuClaw is built upon the [OpenClaw](https://github.com/openclaw/openclaw) open-source project, which provides the core gateway architecture for AI agents.

We are grateful to the OpenClaw community for their pioneering work in AI agent gateways.

## Product Lines

| Product | Description |
|---------|-------------|
| SecuClaw Platform | Core security operations platform |
| SecuHub | Skill marketplace for security capabilities |
| SecuClaw Intelligence | Threat intelligence services |
| SecuClaw Command | Security operations center |
| SecuClaw Compliance | Compliance management |

## Technology Stack

- **Runtime**: Node.js 22+
- **Language**: TypeScript
- **AI**: Anthropic Claude, OpenAI GPT, Ollama
- **Database**: SQLite (JSONL), ChromaDB (vectors)
- **Container**: Docker
